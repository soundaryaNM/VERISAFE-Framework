# VERISAFE UI (static demo)

Lightweight static UI demonstrating the responsive, safety-focused layout for VERISAFE.

Files:
- `index.html` — static markup
- `styles.css` — responsive CSS grid and visual system
- `app.js` — minimal client interactions for demo

Open `index.html` in a browser to view. This is a local-only static demo and intentionally avoids external dependencies.
